## 📊 Технические детали

### Web Speech API
- Язык: **ru-RU** (русский)
- Непрерывность: **отключена** (одна фраза за раз)
- Браузеры: Chrome, Edge, Safari (iOS 14.5+), Opera
- Firefox: требует отдельного решения (не поддерживается пока)

### Производительность
- Нет новых npm-пакетов
- Нет увеличения бандла (используется встроенный API браузера)
- Сборка: **0ms** (Next.js оптимизация)
- Size: стабилен (~107 kB First Load JS)

### Безопасность
- Браузер требует явное разрешение на доступ к микрофону
- HTTPS требуется для полной функциональности
- Нет отправки данных на сторонние серверы (Web Speech API Google работает в браузере)

## ⚠️ Ограничения первой версии

1. **Браузерная поддержка** — не работает в Firefox (требует доп. решения)
2. **Точность** — зависит от качества микрофона и шума
3. **Интернет** — требуется соединение
4. **Мобила** — нужно явное разрешение на микрофон
5. **Озвучивание** — ответы только текстом (озвучивание позже)
6. **История** — голосовые входы не сохраняются
