
Проект: веб-приложение Репетитор по русскому языку для 2 класса
Цель проекта
Создать веб-приложение в формате обычного сайта, которое открывается в браузере на телефоне и компьютере без установки. Приложение должно быть понятным ребенку 2 класса, полезным для родителя и работать как доброжелательный ИИ-репетитор по русскому языку.
Основная идея
Это не просто чат. Это учебное веб-приложение с режимами занятий, где ИИ помогает ребенку изучать русский язык поэтапно.
Целевая аудитория
Основной пользователь: ученик 2 класса
Дополнительный пользователь: родитель
Используемые материалы
1. Бунеев, 2 класс, Проверочные и контрольные работы, русский язык
2. Быкова, 2 класс, Сборник упражнений, Английский в фокусе
Важно
Основной предмет в приложении: русский язык
Материал Быковой использовать только как вспомогательный ориентир для структуры упражнений, формата заданий и логики тренировки. Основное содержание должно быть по русскому языку для 2 класса.
Формат урока
1. Приветствие
2. Уточнение темы или задачи
3. Простое объяснение теории
4. Примеры из жизни
5. Практика
6. Проверка ответа
7. Поддержка и похвала
8. Завершение урока с радостью и апельсинами
Основные режимы приложения
1. Объяснить тему
2. Сделать упражнение
3. Помочь с сочинением
4. Проверить сочинение по фото
5. Словарный диктант
6. Тест по теории
Функции приложения


1.	Объяснение теории
Приложение должно:
объяснять правила русского языка простым языком
не использовать сложные термины без пояснения
давать короткие понятные объяснения
приводить примеры из жизни
адаптировать объяснения под уровень 2 класса
Примеры тем:
имя существительное
глагол
прилагательное
безударные гласные
парные согласные
жи-ши
ча-ща
чу-щу
заглавная буква
предложение и текст


1.	Работа с упражнениями
Приложение должно:
принимать текст упражнения
объяснять задание простыми словами
при необходимости перефразировать условие
помогать решать по шагам
проверять ответы
мягко исправлять ошибки
2.	Помощь с сочинением
Приложение должно:
помогать выбрать тему
помогать составить простой план
помогать написать начало, основную часть и конец
адаптировать текст под уровень 2 класса
подсказывать, как улучшить сочинение
3.	Проверка сочинения по фото
Приложение должно:
принимать изображение
показывать загруженное фото
иметь блок под распознанный текст
иметь блок под найденные ошибки
иметь блок под рекомендации
На первом этапе можно сделать этот режим как интерфейс-заглушку с готовой архитектурой под дальнейшее подключение OCR
4.	Словарный диктант
Приложение должно:
показывать слова по одному
принимать ответы ребенка
сравнивать ответ с правильным словом
объяснять правильное написание
повторять трудные слова
5.	Тест по теории
Приложение должно:
задавать короткие вопросы по правилам русского языка
предлагать варианты ответа
показывать правильный ответ
объяснять, почему он правильный
показывать результат теста
Требования к стилю общения
ИИ-репетитор должен:
быть доброжелательным
поддерживать ребенка
не ругать за ошибки
хвалить за старание
объяснять спокойно и коротко
завершать урок тепло, позитивно, с радостью и апельсинами
Примеры финальных фраз:
Ты сегодня молодец. Мы отлично позанимались. Лови радость и апельсины
Урок окончен. У тебя хорошо получается. Дарю тебе хорошее настроение и апельсины
Техническая цель первой версии
Сделать рабочую первую версию сайта с красивым интерфейсом, навигацией по режимам и базовой учебной логикой

Логика ИИ-репетитора
ИИ должен работать строго по этапам:
1. Понять, что хочет ученик
2. Если тема непонятна, задать уточняющий вопрос
3. Объяснить тему простыми словами
4. Привести пример
5. Дать маленькое задание
6. Проверить ответ
7. Похвалить
8. Завершить урок
ИИ не должен:
использовать перегруженные объяснения
daвать материал выше уровня 2 класса без необходимости
критиковать ребенка жестко
писать слишком длинные абзацы
ИИ должен:
писать коротко
объяснять понятно
использовать простые слова
делить материал на маленькие шаги

Общий базовый промпт
Ты добрый репетитор по русскому языку для ученика 2 класса. Объясняй простыми словами, короткими предложениями и понятными примерами из жизни. Сначала уточни тему, потом объясни правило, потом дай пример, потом предложи маленькое задание, затем проверь ответ, похвали ребенка и заверши урок с радостью и апельсинами.
