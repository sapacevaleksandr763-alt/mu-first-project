### 2. Создан компонент голосового ввода
**Файл:** `components/VoiceInput.tsx`

Компонент реализует:
- **Web Speech API** для распознавания речи на русском языке (`ru-RU`)
- **Состояния интерфейса:**
  - `ready` — готов к записи (кнопка 🎤)
  - `recording` — идёт запись (кнопка 🎙️ Слушаю...)
  - `processing` — обработка распознавания (кнопка ⏳ Жди...)
  - `error` — ошибка (кнопка ❌ Ошибка)
  - `unsupported` — браузер не поддерживает Web Speech API

- **Обработка ошибок:**
  - Браузер не поддерживает Web Speech API → показ информационного сообщения
  - Пользователь не дал разрешение на микрофон → понятное сообщение об ошибке
  - Сетевые ошибки → обработка и вывод сообщения
  - Отсутствие звука → подсказка попробовать ещё раз

- **Callback:** `onVoiceResult(text: string)` для передачи распознанного текста родительскому компоненту

## 🎯 Как это работает

### Сценарий использования:
1. Ребёнок видит поле ввода и кнопку 🎤
2. Нажимает на кнопку микрофона
3. Браузер запрашивает разрешение на доступ к микрофону (если требуется)
4. Кнопка меняется на 🎙️ Слушаю... (красная с анимацией)
5. Ребёнок говорит вопрос или ответ на русском языке
6. Система распознаёт речь и конвертирует в текст
7. Текст автоматически подставляется в поле ввода
8. Кнопка меняется на ⏳ Жди... (оранжевая)
9. Ребёнок может нажать кнопку "Отправить" или просто ввести ещё текст
10. Ответ ИИ приходит текстом на экран

### Обработка ошибок:
- **Браузер не поддерживает:** Кнопка скрывается, показывается сообщение "Твой браузер не поддерживает голосовой ввод"
- **Нет разрешения на микрофон:** Показывается сообщение "Разрешение на микрофон отклонено"
- **Сетевая ошибка:** Показывается сообщение "Ошибка сети"
- **Не услышал голос:** Показывается "Я не услышал голос. Попробуй ещё раз"

## 🔧 Технические детали

### Web Speech API
- **Поддержка браузерами:** Chrome, Edge, Safari (iOS 14.5+), Opera
- **Не поддерживают:** Firefox (требует специального API)
- **Язык:** ru-RU (русский)
- **Непрерывная речь:** отключена (каждый клик = одно распознавание)

### Обработка распознавания
- Система собирает промежуточные результаты (`interimResults`)
- Финальный результат отправляется через callback
- Текст обрезается от пробелов

### Состояния для UX
- Каждое состояние имеет свой цвет и иконку
- Анимация пульса при записи (приближает к реальности)
- Отключение кнопки при обработке (предотвращает двойные клики)

## ⚠️ Ограничения первой версии

1. **Браузерная поддержка:** Web Speech API работает не во всех браузерах (Firefox нужно отдельное решение)
2. **Точность распознавания:** Зависит от качества микрофона и шума в окружении
3. **Интернет:** Требует интернет-соединение (функция работает online)
4. **Язык:** Фиксирован на русский (можно расширить на другие языки)
5. **Озвучивание:** Ответы ИИ выводятся только текстом (озвучивание можно добавить позже)
6. **Аккаунты:** Нет сохранения истории микрофонных входов
7. **Мобила:** На мобильных устройствах необходимо явное разрешение на доступ к микрофону
