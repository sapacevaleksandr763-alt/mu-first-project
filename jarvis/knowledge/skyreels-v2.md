# SkyReels V2 — генерация видео через ИИ

> Источник: гайд Дмитрия Ледовских (@mcdenil), май 2025

## Что это

Бесплатная open-source программа для генерации видео по тексту или изображению. Без подписок, без водяных знаков.

**Возможности:**
- Текст → Видео
- Картинка → Видео (оживление фото)
- Продление видео (удлинение клипа)
- Бесконечная длина (от 4 сек до нескольких минут)
- Разрешение: 540P (544×960) и 720P HD (720×1280)

## Две версии

| Версия | VRAM | Для кого |
|--------|------|----------|
| **1.3B** (лёгкая) | от 14.7 ГБ | 90% задач — рилсы, прогревы, контент |
| **14B** (полная) | от 43 ГБ | Профессиональный уровень, серверное железо |

## Вариант 1: Свой ПК

```bash
# Требуется: Python 3.10, Git
git clone https://github.com/SkyworkAI/SkyReels-V2
cd SkyReels-V2
pip install -r requirements.txt
python inference.py --prompt "your prompt here"
```
