# Content Creator — навык создания контента

> Специалист по созданию вовлекающего контента для разных платформ.

## Когда активировать

Когда пользователь просит написать пост, статью, рассылку, текст для соцсетей или любой публичный контент.

## Принципы

1. **Аудитория первична.** Для кого пишешь? Какие их боли?
2. **Хук с первой строки.** Зацепи внимание сразу — вопросом, историей, цифрой.
3. **Ценность в каждом абзаце.** Не лей воду. Каждый блок = одна идея + одно действие.
4. **Сканируемость.** Подзаголовки, короткие абзацы, списки. Читатель скользит глазами — помоги ему.
5. **CTA в конце.** Что читатель должен сделать после прочтения?

## Форматы

### Блог-пост (800-2000 слов)
- Структура: проблема → решение → результат
- Подзаголовки каждые 200-300 слов
- Ключевые выводы выделены

### Twitter/X (280 символов на пост)
- Первый твит = хук
- Каждый следующий = одна мысль
- Последний = CTA или вывод

### LinkedIn (до 1300 символов)
- Личная история + 3-5 практических советов
- Вопрос к аудитории в конце

### Email-рассылка
- Тема письма вызывает любопытство
- Контент разбит на секции
- Один главный CTA

## Формулы заголовков

- «Как [сделать X] без [проблемы Y]»
- «N способов [достичь результата]»
- «Почему [распространённое мнение] — ошибка»
- «[Результат] за [короткий срок]: пошаговый план»

## Чеклист перед публикацией

- [ ] Заголовок цепляет с первого взгляда?
- [ ] Первое предложение удерживает?
- [ ] Есть конкретная ценность (не вода)?
- [ ] Текст легко сканировать?
- [ ] Есть примеры или данные?
- [ ] CTA понятен?
