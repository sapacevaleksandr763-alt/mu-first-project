# Структура сайта — Славяно-Арийская Культура

## Файловая структура проекта

```
aleksandr-sapachev-setup/
│
├── index.html                 ← посадочная страница (готова)
│
├── css/
│   └── style.css              ← все стили
│
├── js/
│   └── main.js                ← поведение (бургер, анимации, форма)
│
├── images/
│   ├── backgrounds/           ← фоны секций
│   ├── gallery/               ← фото обрядов, артефактов, мест
│   └── icons/                 ← иконки символов
│
├── fonts/                     ← если понадобятся локальные шрифты
│
└── pages/                     ← страницы для будущего сайта
    ├── history.html           ← История и хронология
    ├── gods.html              ← Полный пантеон богов
    ├── traditions.html        ← Традиции и обряды
    ├── runes.html             ← Руны — система и значения
    ├── vedas.html             ← Веды и тексты (Велесова книга)
    └── gallery.html           ← Галерея
```

---

## Что на посадочной странице (index.html)

Шесть секций в одном файле:

1. **Hero** — заголовок с коловратом, анимация
2. **О культуре** — вводный текст + 4 карточки
3. **История** — таймлайн от 40 000 л. до н.э. до наших дней
4. **Пантеон богов** — 8 карточек: Сварог, Перун, Велес, Даждьбог, Макошь, Лада, Стрибог, Ярило
5. **Традиции** — годовой круг: Коляда, Масленица, Купала, Осенние деды
6. **Символы** — Коловрат, Громовик, Звезда Сварога, Мировое Древо и др.

## Страницы (pages/)

| Файл | Содержание |
|------|-----------|
| `history.html` | Полная история: трипольская культура, ведический период, расцвет Руси |
| `gods.html` | Расширенный пантеон: старшие и младшие боги, их атрибуты и обряды |
| `traditions.html` | Все обряды и традиции: свадебные, погребальные, земледельческие |
| `runes.html` | Рунический алфавит, значения знаков, применение в обережах |
| `vedas.html` | Велесова книга, Риг-Веда, Авеста — тексты и их толкование |
| `gallery.html` | Фото: капища, артефакты, символы, природные места силы |

## Дизайн

- Цветовая схема: тёмная + золото (`#c9a84c`)
- Шрифты: Cinzel (заголовки) + Raleway (текст)
- Адаптив: мобильные, планшеты, десктоп
- Анимации: появление при скролле, вращение коловрата
