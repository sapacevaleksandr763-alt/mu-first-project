# Наследие Ариев — Сайт о славяно-арийской культуре

## Что это

Сайт **«Наследие Ариев»** для Александра Сапачёва — авторский проект о славянской культуре, ведическом наследии, богах, рунах, традициях и обрядах. Цель — стать лучшим сайтом по теме на русскоязычном рынке.

## Технологии

- Чистый HTML/CSS/JS (без фреймворков)
- Шрифты: Cinzel (заголовки) + Raleway (текст) — Google Fonts

## Как работать с проектом

1. Файлы редактируются **локально** (VS Code Claude Code)
2. Загружаются на сервер через: `scp файл root@155.212.208.32:/home/jarvis/projects/aleksandr-sapachev-setup/путь`
3. **Большие файлы** (>100 строк) — правь точечно через `sed` на сервере, не перезаписывай целиком
4. Стили — в `css/style.css` (общие) и `css/pages.css` (подстраницы)
5. Интерактивные страницы содержат инлайн-стили в `<style>` и инлайн-скрипты — это нормально, они самодостаточные
